Reading The HSB Color System: A Practitioner's Primer by Erik D. Kennedy inspired me to put toghether this visual and practical guide, somewhere to have some hands-on practice on this great color system.

HUE

Hue is measured in degrees, so it is a number between 0 and 360:
Pure red is 0° (that is the same to 360°, the color wheel is a circle, remember?), pure green is 120° and pure blue is 240°.

LIGHTNESS

Sometimes refered to as "Brightness", it is a number between 0 and 100, it controls how light or dark is a color:
0% lightness is black and 100% lightness is white. 50% lightness is the color in its original brightness.
